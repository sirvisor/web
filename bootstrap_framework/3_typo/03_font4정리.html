3.small, strong, em 태그의 사용법과 .lead 클래스 사용

small태그는 작은 글씨를, strong은 문장을 강조할 때, em태그는 이탤릭체로 표시하게 한다.

리드(.lead) 클래스 선택자를 이용하여 단락의 첫 문장을 강조할 수 있는데, 글자의 크기는 16px의 살짝 두꺼운 글꼴로 표시된다.

4. 문장의 정렬은 .text-left, .textcenter, .text-right 클래스 선택자를 사용한다

왼쪽 정렬은 .text-left 클래스 선택자를 사용한다. 왼쪽은 기본이기 때문에 다른 문장이 가운데 정렬이거나, 오른쪽 정렬일 때 왼쪽으로 정렬하고 싶은 경우 사용한다.

가운데 정렬은 .text-center 클래스 선택자를 선택한다

오른쪽 정렬은 .text-right 클래스 선택자를 사용한다.

6. 약어를 사용할 수 있다.

약어를 사용하면 멘붕과 같은 약어를 표현할 수 있다.

  • 약어 사용시 점선으로 밑줄이 그어진다.
  • 해당 단어에 마우스를 올리면 물음표가 나오면서 원래 텍스트가 노출된다.
  • un 와 같은 영문 약어는 class="initialism"을 사용하게 되면 단어가 소문자인 경우에도 대문자로 자동 변경된다.
  • 글씨 크기는 90% 정도로 조정된다.
